# Customer Order and Retention Analysis 
## Project Overview 
This project analyzes customers order behavior to understand customer segments, repeat orders, spending behavior, satisfaction, also retention signals. 
## Business Objective
The objective is to identify which customer segments, regions, channels order, product categories, and also delivery status impact to customer order value and satisfaction, while also documenting data quality issues that should be reviewed before making final decisions from insight dashboard. 
## Dataset 
The dataset contains customer orders activites records with fields such as customer name, order date, region, acquisition channel, product category, items purchased, order value, delivery status, satisfaction, use of promotion.
